# puppeteer
用chrome的puppeteer(无头浏览器)做的小米商城数据爬虫


##### 文件目录介绍
```js
|-- puppeteer
    |-- .DS_Store    
    |-- .gitignore
    |-- LICENSE
    |-- README.md
    |-- detail.json // 小米商城详情数据
    |-- mi-data.js //爬取小米商城详情数据
    |-- package-lock.json
    |-- package.json
    |-- sobooks.js //爬取sobooks数据
    |-- test.js //puppeteer测试
    |-- 小米商城数据
        |-- .DS_Store 
        |-- newData.js //操作有用的数据
        |-- newData.json //有用的数据
        |-- newPro.json //将所有数据合并成一个数组
        |-- oldData.js //操作原始数据
        |-- oldData.json //原始数据
```
